技术栈详情

🎨 前端技术

  • React 18 - 组件化开发
  • Redux Toolkit - 状态管理
  • React Router - 路由管理
  • Axios - HTTP 请求
  • Tailwind CSS - 样式框架
  • 响应式设计 - 移动端适配

⚙️ 后端技术

  • Node.js + Express - 服务器
  • RESTful API - 接口设计
  • JWT - 身份认证
  • Stripe API - 支付集成
  • Multer - 文件上传
  • Helmet - 安全加固

🗄️ 数据库

  • MongoDB - 文档数据库
  • Mongoose - ODM 框架
  • Redis - 缓存层
  • GridFS - 大文件存储

🚀 架构亮点

  • 微服务架构设计
  • JWT 无状态认证
  • 图片懒加载优化
  • API 速率限制
  • 错误日志监控
  • Docker 容器化部署
